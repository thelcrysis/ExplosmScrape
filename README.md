# ExplosmScrape
Scraping all comics from Cyanide & Humor's website, ie explosm.net. About 5k comics exist and this Python script will collect them all inside a single 'output.html' file. 

## How to run
from bash
$python scraper.py

## Dependencies
urllib2
bs4' BeautifulSoup
sys
os
